[HandBrake](https://github.com/HandBrake/HandBrake) is an open-source, GPL-licensed, multi-platform video transcoder.

## Install

### The canonical Handbrake

Download and run the installer at: <https://handbrake.fr/>

Or install via chocolatey

```powershell
choco install handbrake
```

### For the `SVT-AV1-HDR` encoder

[This fork](https://github.com/Uranite/HandBrake-SVT-AV1-HDR) replaces the SVT-AV1 encoder with [SVT-AV1-HDR](https://github.com/juliobbv-p/svt-av1-hdr) inside HandBrake.
