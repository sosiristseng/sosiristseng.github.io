[nvtop](https://github.com/Syllo/nvtop) is a GPU process monitor for AMD, Intel and NVIDIA GPUs.

## Installation

### Ubuntu

```sh
sudo apt install nvtop
```

### Arch Linux

```sh
sudo pacman -S nvtop
```

## Usage

Run `nvtop`.
