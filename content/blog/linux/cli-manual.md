Alternatives to the [man](https://linux.die.net/man/) command:

<!--more-->

## cheat.sh

[cheat.sh](https://cheat.sh/) is an online cheat sheet that can be also accessed via the `curl` command.

For example,

```sh
curl cht.sh/tar
```

## tldr

[tldr](https://tldr.sh/) are collaborative cheat sheets for console commands, a complement to `man` pages.
