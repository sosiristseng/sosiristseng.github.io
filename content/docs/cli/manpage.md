`man` is the command for manual pages. See the usage a command by running `man <cmd>` in the terminal.

There are also cheat sheets with a brief example for a command.

## cheat.sh

[cheat.sh](https://cheat.sh/) is an online cheat sheet that can be also accessed via the `curl` command.

For example,

```sh
curl cht.sh/tar
```

## tldr

[tldr](https://tldr.sh/) are collaborative cheat sheets for console commands, a complement to `man` pages.

Go to [tldr's InBrowser.App](https://tldr.inbrowser.app/) in a web browser.

## tealdeer

[tealdeer](https://github.com/tealdeer-rs/tealdeer) is a fast implementation of `tldr` in Rust. `tldr` are simplified, example based and community-driven man pages.
