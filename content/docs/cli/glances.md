[glances](https://github.com/nicolargo/glances) is a cross-platform advance system monitor for usage monitoring for CPU, RAM, Network, GPU, etc.

The simplest way to run `glances` is to use `uv`:

```sh
## Update every 1 second
uvx glances -T1
```
