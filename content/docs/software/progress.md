[progress](https://github.com/Xfennec/progress) measures the speed and progress of ongoing commands (For instance, file copy command `cp`).

<!--more-->

## Install

```sh
sudo apt install progress
```

## Usage

[Run `progress`](https://github.com/Xfennec/progress#what-can-i-do-with-it) in the terminal.
