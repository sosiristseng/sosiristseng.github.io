[pstop](https://github.com/psmux/pstop) is a HTOP-like TUI system monitor for Windows.

## Install

**winget**

```powershell
winget install marlocarlo.pstop
```

**Chocolatey**

```powershell
choco install pstop
```

## Run

```powershell
pstop
```

or

```powershell
htop
```
